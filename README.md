### 🧠 What I actually do

- 🛰️ **Edge AI on UAV platforms** — IR/EO payloads, real-time target detection, the kind of latency budgets that make cloud engineers cry
- 🎯 **Classical CV when it's smarter** — warp-and-diff motion detection beats a 200MB model when you have 40ms and no training data
- ⚡ **Jetson whispering** — AGX Orin, Thor, anything with CUDA and a thermal problem
- 🎓 **Teaching cloud computing** at Atılım University — yes, I feel the irony of teaching cloud while deploying to devices with 32GB of RAM
- 🤖 **LLMs where they earn their keep** — tactical reasoning, doc extraction, and occasionally roasting my own code
